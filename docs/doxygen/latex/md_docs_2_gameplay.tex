\hypertarget{md_docs_2_gameplay_autotoc_md143}{}\doxysection{\texorpdfstring{Enemies}{Enemies}}\label{md_docs_2_gameplay_autotoc_md143}
Enemies should typically have\+:


\begin{DoxyItemize}
\item Movement/behavior updates per frame ({\ttfamily tick})
\item A way to take damage (health decreases)
\item A state that determines whether they’re active/alive
\end{DoxyItemize}

Possible next behaviors\+:
\begin{DoxyItemize}
\item Patrol in an area
\item Chase player within a detection radius
\item Stop/idle when far away
\item Attack player when close
\end{DoxyItemize}
