\doxysubsection{Detailed Description}
AI-\/controlled enemy that chases and damages the player. 

\doxylink{class_enemy}{Enemy} uses a target \doxylink{class_character}{Character} to\+:
\begin{DoxyItemize}
\item Compute screen position relative to the player world position.
\item Move towards the player until within a stop radius.
\item Apply continuous damage while colliding with the player. 
\end{DoxyItemize}
