\hypertarget{md_docs_2_assets_autotoc_md47}{}\doxysection{\texorpdfstring{Naming Conventions (Recommended)}{Naming Conventions (Recommended)}}\label{md_docs_2_assets_autotoc_md47}

\begin{DoxyItemize}
\item Use lowercase with underscores\+:
\begin{DoxyItemize}
\item {\ttfamily knight\+\_\+idle.\+png}
\item {\ttfamily weapon\+\_\+sword.\+png}
\item {\ttfamily enemy\+\_\+slime\+\_\+walk.\+png}
\end{DoxyItemize}
\item Keep folders descriptive\+:
\begin{DoxyItemize}
\item {\ttfamily characters/}
\item {\ttfamily tiles/} or {\ttfamily nature\+\_\+tileset/}
\item {\ttfamily ui/}
\end{DoxyItemize}
\end{DoxyItemize}

\hypertarget{md_docs_2_assets_autotoc_md49}{}\doxysection{\texorpdfstring{Adding a New Asset}{Adding a New Asset}}\label{md_docs_2_assets_autotoc_md49}

\begin{DoxyEnumerate}
\item Place the file under {\ttfamily assets/} in an appropriate folder
\item Reference it from code via a constant/path define (or {\ttfamily constexpr})
\item Load it using raylib\+:
\begin{DoxyItemize}
\item {\ttfamily Load\+Texture("{}assets/..."{})}
\end{DoxyItemize}
\end{DoxyEnumerate}

{\bfseries{Tip\+:}} Prefer storing paths in one place so you don’t duplicate strings.

\hypertarget{md_docs_2_assets_autotoc_md51}{}\doxysection{\texorpdfstring{Credits / Licensing}{Credits / Licensing}}\label{md_docs_2_assets_autotoc_md51}
If you use third-\/party art/audio, keep a record of\+:


\begin{DoxyItemize}
\item Author/creator
\item License type (CC0, CC-\/\+BY, etc.)
\item Source location
\end{DoxyItemize}

Recommended\+: add a {\ttfamily \doxylink{_credits_8md}{CREDITS.\+md}} file or a “\+Credits” section in {\ttfamily \doxylink{_r_e_a_d_m_e_8md}{README.\+md}}.

\begin{DoxyItemize}
\item Load textures once and reuse them
\item Avoid loading textures inside {\ttfamily tick()}; load at initialization or in a resource manager
\item Consider unloading textures on shutdown (or when changing levels)
